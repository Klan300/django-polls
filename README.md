# Django Polls Application
## Feature

- **Admin**
    - add polls
    - delete polls
    - check polls status
- **Polls**
    - vote polls
    - check status polls vote

 ## Requirements

 The application requires
 * Python 3.6 or newer
 * Django 2.1.2 or newer
 * Python add-on modules as in [requirements.tx](requirements.txt)

 ## Running

1. Dowload **Django**
2. Run **Django**
    ```bash
    python3 manage.py runserver
    ```
